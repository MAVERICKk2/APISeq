 Information
Returns API doc
POST an information message to SEQ

  • URL
    • /Information

  • Method:
    • POST | GET

  • URL Params
    • None

  • Data Params
    • Contet JSON
    {
    "apikey": SEQ Application Token,
    "message": Message,
    "component": Database-.NET-Web Service,
    "ip": source IP,
    "machine": source host,
    "user": source User,
    "location": country-city-etc,
    "os": source OS,
    "browser": source browser,
    "other": other
    }

  • Success Response:
    • Code: 200
      Content {"response":"Your information message was sent"}
  • Error Response
    • Code: 400
      Content {"response":"Provide a valid application Token"}
    • Code: 500
      Content {"response":"SEQ Server is not available!"}
  • Sample Call
    • {
    "apikey": "NFn3QVPYlyOkOhAGydzQ",
    "message": "The End is Near",

    "machine": "PB039EV5",
    "user": "U11231",
    "location": "Spain",
    "os": "Windows",
    "browser": "Firefox",
    "other": ""
    }
